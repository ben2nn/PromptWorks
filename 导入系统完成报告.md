## 🎨 技术架构

### 数据流程

```
JSON 文件
    ↓
[import_prompts.py]
    ↓
解析 → 验证 → 处理
    ↓
标签 → 图片 → 提示词
    ↓
数据库 (PostgreSQL)
```

### 技术栈

- **Python 3.10+**: 主要开发语言
- **SQLAlchemy**: ORM 框架
- **httpx**: HTTP 客户端
- **Pillow**: 图片处理
- **PostgreSQL**: 数据库

### 设计模式

- **单一职责**: 每个脚本专注一个功能
- **错误容忍**: 单个失败不影响整体
- **缓存优化**: 标签缓存避免重复查询
- **事务管理**: 每个提示词独立事务

## 🚀 使用方式

### 快速开始

```bash
# Windows PowerShell (推荐)
.\scripts\example_usage.ps1

# 或手动执行
python scripts/ensure_class.py
python scripts/test_import.py prompts_20251025_192918.json
python scripts/import_prompts.py prompts_20251025_192918.json
```

### 高级用法

```bash
# 指定分类
python scripts/import_prompts.py data.json 5

# 批量导入
Get-ChildItem *.json | ForEach-Object {
    python scripts/import_prompts.py $_.Name
}
```

### 场景1: 我是新手，第一次使用

**推荐阅读顺序:**

**预计时间**: 15-20分钟

### 场景2: 我遇到了问题

**故障排查路径:**

### 场景3: 我想了解技术细节

**深入学习路径:**
1. [IMPORT_SUMMARY.md](IMPORT_SUMMARY.md) - 技术架构

3. 查看源代码 `scripts/import_prompts.py`
4. 阅读数据库模型 `app/models/prompt.py`

**预计时间**: 1-2小时

### 场景4: 我想批量导入多个文件

**操作指南:**
1. 查看 [IMPORT_GUIDE.md](IMPORT_GUIDE.md) 的高级用法
2. 使用 PowerShell 批量脚本:
   ```powershell
   Get-ChildItem *.json | ForEach-Object {
       python scripts/import_prompts.py $_.Name
   }
   ```

### 场景5: 我想自定义导入逻辑
